Munus Gesture Privacy Policy

Last updated: April 22, 2026

Munus Gesture is a browser extension that provides mouse gesture controls, left-click and modifier-key gestures, rocker gestures, super drag, site-specific settings, send-key actions, wheel tab switching, and smooth scrolling features.

1. Data Collection and Transmission

Munus Gesture does not collect or transmit personal data to external servers.

Munus Gesture does not send user data to external services for advertising, analytics, profiling, or marketing purposes.

2. Data Processed Locally by the Extension

To provide its features, Munus Gesture may access and process the following information locally in the user's browser:

  • Tab information required for tab-related actions
  • DOM elements on the current webpage such as links, images, selected text, and page URLs
  • User settings such as gesture mappings, site-specific rules, key actions, and display preferences

This information is used locally only to operate the extension's features.

3. Stored Information

Munus Gesture stores user settings in Chrome extension storage.

  • Gesture mappings
  • Modifier and rocker gesture settings
  • Site-specific settings and saved rules
  • Send-key settings
  • Display and behavior settings required for the extension to operate properly

4. Third-Party Sharing

Munus Gesture does not sell, share, or provide user data to third parties.

5. Remote Code

Munus Gesture does not download or execute remotely hosted code. It uses only code bundled in the published extension package.

6. Changes to This Policy

This Privacy Policy may be updated if the extension's features or behavior change. Any updates will be posted on this page.
