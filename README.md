# Munus Gesture

Munus Gesture is a Chrome extension designed to improve browsing efficiency with flexible and advanced mouse gesture controls.

## Features

- Standard mouse gestures
- Left-click + gesture
- Modifier-key + gesture
- Rocker gestures
- Super Drag for links, images, and text
- Site-specific gesture settings
- Send-key actions
- Wheel tab switching
- Smooth scroll for gesture actions
